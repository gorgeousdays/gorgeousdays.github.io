OD基本指令

指令 快捷键 含义
Restart Ctrl+F2 重新开始调试
Step Into F7 执行一行OP code，若遇CALL 将进入函数代码内部
Step over F8 执行一行OP code，若遇CALL尽执行函数自身，不跟随
Execute till Return Ctrl+F9 一直在函数代码内部运行，直到遇到RETN，跳出函数
Go to Ctrl+G 移动到指定地址(运行时不可用)
Execute till Cursor F4 执行到光标位置，即直接转到要调试的地址
Comment ; 添加注释
Label : 添加标签
Set/Reset BreakPoint F2 设置/取消断点
Run F9 运行(若设置了端点，则执行至断点处)
Show the current EIP * 显示当前EIP位置
Show the previous Cursor - 显示上一个光标的位置
Preview CALL/JMP address Enter 若光标处有CALL/JMP指令，则跟踪并显示相关地址(运行时不可用)
Edit data Ctrl+E 编辑数据
Assemble Space 编写汇编代码

汇编指令

指令 说明
CALL xxx 调用xxx处的函数
JMP xxx 调整至xxx地址处
PUSH xxx 保持xxx到栈
RETN 跳转到栈中保持的地址
